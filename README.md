# React Chatting UI 

React와 TailwindCSS로 개발한 Chatbot UI 입니다.
 <br /> <br /> <br />
## 시작하기 (Getting Started)

1. 리포지토리를 클론합니다. <br />
`git clone https://github.com/your-project.git`

2. 필요한 npm 패키지를 설치합니다. <br />
`npm install`

3. 프로젝트를 실행합니다. <br />
`npm run start`
